
# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
Understand how to read, summarize, and visualize a dataset using pandas, matplotlib, and seaborn.

## Instructions Completed
- ✅ Load the dataset using the pandas library
- ✅ Display dataset structure using .shape, .columns, and .head()
- ✅ Create basic visualizations:
  - ✅ Scatter plot to analyze relationships between variables
  - ✅ Histogram to examine data distribution  
  - ✅ Box plot to detect outliers and spread of values
- ✅ Use matplotlib and seaborn for visualizations

## Skills Demonstrated
- Data loading and inspection using pandas
- Basic data summarization
- Visualization using matplotlib and seaborn

## Dataset
- **Name**: Iris Dataset
- **Source**: Loaded from seaborn built-in datasets
- **Samples**: 150
- **Features**: 4 numerical (sepal_length, sepal_width, petal_length, petal_width)
- **Target**: Species (setosa, versicolor, virginica)

## Code Structure
1. **Data Loading**: Load Iris dataset using seaborn
2. **Data Inspection**: Use .shape, .columns, and .head() methods
3. **Data Summarization**: Basic statistics and info
4. **Visualization**:
   - Scatter plots for feature relationships
   - Histograms for data distributions
   - Box plots for outliers and value spreads

## Requirements
- Python 3.x
- pandas
- matplotlib
- seaborn
