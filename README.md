* **Objective:** Predict if a customer will subscribe to a term deposit following a marketing campaign.

* **Methodology:** Conducted EDA and applied K-Means Clustering.
* **Visualization:** Used PCA or t-SNE for dimensionality reduction and cluster visualization.

* **Objective:** Forecast household energy usage using historical patterns.
* **Methodology:** Resampled time series data and engineered features like hour of day and weekday/weekend.

* **Metrics:** Evaluated results using MAE and RMSE.
